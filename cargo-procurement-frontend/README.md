# Cargo Procurement Frontend

React frontend for the Cargo + Procurement + Invoice Management System.

## Features

- Login and registration
- Dashboard summary
- Customer management
- Supplier management
- Procurement requests
- Shipment creation and status updates
- Public shipment tracking page
- Invoice creation, payment update and PDF download
- Expense tracking
- Document upload for shipments

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

Default API URL:

```txt
http://localhost:5000/api
```

Make sure the backend is running on port `5000`.

## Login

Create an admin account from the Register page, then login.

## Public Tracking Page

Open:

```txt
http://localhost:5173/track
```

Enter a shipment tracking number generated by the backend.
